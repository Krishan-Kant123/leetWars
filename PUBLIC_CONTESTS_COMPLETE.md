## рҹҺҜ How It Works

### Creating Contests:

**Public Contest:**
1. Go to Create Contest
2. Select "рҹҢҚ Public"
3. Fill details + select problems
4. Create!
5. **Visible to everyone on dashboard**

**Private Contest:**
1. Go to Create Contest
2. Select "рҹ”’ Private" (default)
3. Fill details + select problems
4. Create!
5. **Only accessible via code**

### Joining Contests:

**Public Contests:**
```

```


## рҹ”җ Security

**Public Contests:**
- вң… Visible to all authenticated users
- вң… Can enroll without code
- вң… Listed in `/public/all` endpoint
- вң… Shows enrollment status (`isEnrolled`)

**Private Contests:**
- вң… NOT visible in public list
- вң… CANNOT enroll via ID (returns 403 Forbidden)
- вң… Only accessible with contest code
- вң… Not shown to unenrolled users
